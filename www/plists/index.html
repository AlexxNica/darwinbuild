Property Lists

Overview

DarwinBuild is capable of building projects found in many Darwin releases. Each release is described by a property list, which DarwinBuild uses to perform its builds. Property lists for supported releases can be found below. If you need support for a release not mentioned here, please email the mailing list.

Releases

Each Mac OS X version has a unique build number. Darwin property lists with this build number describe the various open source projects and versions in that corresponding Mac OS X version, as they are published by Apple. OpenDarwin property lists contain additional information, such as patches that are necessary to build the open source projects on a pure-Darwin system.
